CDM (CASH DEPOSIT MACHINE) — REFERENCE NUMBER GUIDE

If you are paying via CDM, please capture a photo of the slip at the time of upload to the LMS. Follow these instructions to find your Reference Number:

Your Reference Number consists of two parts:
  • Trace ID or Record NO — a 4-digit number from your slip
  • Terminal ID or Station ID — an English letter code from your slip

Combine both to create your Reference Code:

Example — Trace/Record NO: 2770
Example — Terminal/Station ID: AECKSB1
Combined Reference Code: 2770AECKSB1

See the example CDM receipts below to identify where these numbers are located:
